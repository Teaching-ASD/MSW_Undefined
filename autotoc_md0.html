"monsters" : " Fallen.json Fallen.json Zombie.json Fallen.json Fallen.json Zombie.json Fallen.json Blood_Raven.json "
}

Ezek a fájlok ugyanilyen felépítésű json fileok ### Pl.

{

A program a következő kulccsokat fogadja el: name, *health_points , *damage, *attack cooldown, experience_per_level, health_point_bonus_per_level, damage_bonus_per_level, cooldown_multiplier_per_level, lore, additional_info, race
